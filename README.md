# LearnLab: Your Personal AI Learning Environment

LearnLab is a comprehensive, agentic platform designed to help you master new skills, track your progress, and stay updated with the latest research. It combines multiple specialized AI agents into a unified interface.

## üåü Features

### ü§ñ Agent Hub
- **AI Tutor**: Interactive chat for learning concepts, debugging code, and getting explanations. Includes a specialized **Code Generator** mode.
- **Curriculum Planner**: Generates personalized week-by-week learning plans based on your goals, skill level, and available time.
- **Quiz Master**: Automatically generates quizzes from your learning material to test your knowledge.
- **Deep Researcher**: Searches ArXiv and the Web to create comprehensive summaries on any topic.
- **Knowledge Base**: Ingests documents (PDFs, Text), URLs, and Sitemaps into a private vector database (RAG).

### üõ†Ô∏è Core Capabilities
- **Dashboard**: Track your progress, streaks, and upcoming modules.
- **Jobs System**: Background processing for large ingestion tasks with real-time status tracking.
- **Feed**: A curated feed of research summaries and updates.
- **Settings**: Manage API keys (BYO OpenAI/Anthropic), notification preferences, and schedules.
- **Social Integration**: Draft and publish learning updates to LinkedIn/Twitter (via webhooks).

## üöÄ Architecture

### Backend (FastAPI + Python)
- **Orchestrator**: Manages agent routing and state.
- **Agents**: Specialized modules for Planning, Quizzing, Research, and Knowledge.
- **RAG Engine**: ChromaDB + OpenAI Embeddings for document retrieval.
- **Caching**: Redis-based caching for high-performance LLM responses.
- **Background Jobs**: RQ (Redis Queue) for async processing.

### Frontend (React + Vite)
- **Modern UI**: Tailwind CSS, Lucide Icons, and a responsive Sidebar layout.
- **Interactive Components**: Timelines, Code syntax highlighting, Markdown rendering.

## üì¶ Setup & Installation

1. **Prerequisites**:
   - Docker & Docker Compose
   - OpenAI API Key

2. **Environment Variables**:
   Copy `.env.example` to `.env` and `.env.development`:
   ```bash
   OPENAI_API_KEY=sk-...
   # Optional:
   ANTHROPIC_API_KEY=sk-ant-...
   ```

3. **Run with Docker**:
   ```bash
   docker-compose up --build
   ```

4. **Access**:
   - Frontend: http://localhost:80
   - Backend API: http://localhost:8000/docs

## üìù Usage

1. **Create a Plan**: Go to "Planner" and describe your learning goal (e.g., "Learn Rust in 4 weeks").
2. **Start Learning**: Click "Start Learning" on a module to chat with the Tutor.
3. **Research**: Use the "Researcher" to deep dive into specific topics.
4. **Test**: Take a quiz generated by the "Quiz Master".
5. **Track**: Watch your progress on the Dashboard.
